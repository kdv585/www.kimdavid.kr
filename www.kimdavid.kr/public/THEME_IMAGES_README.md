# 테마 배경 이미지 추가 가이드

## 필요한 이미지 파일

다음 이미지 파일들을 `public` 폴더에 추가해주세요:

1. **kuromi-bg.png** - 쿠로미 테마 배경 이미지
2. **hellokitty-bg.png** - 헬로키티 테마 배경 이미지  
3. **hangyodong-bg.png** - 한교동 테마 배경 이미지

## 이미지 권장 사양

- **크기**: 1920x1080 이상 (배경으로 사용)
- **형식**: PNG (투명 배경 권장)
- **용량**: 500KB 이하 권장 (로딩 속도 고려)

## 이미지 추가 방법

1. 제공받은 이미지 파일을 `www.kimdavid.kr/public/` 폴더에 복사
2. 파일명을 정확히 맞춰주세요:
   - `kuromi-bg.png`
   - `hellokitty-bg.png`
   - `hangyodong-bg.png`
3. 브라우저 새로고침 후 테마 전환하여 확인

## 이미지가 없을 때

이미지가 없어도 테마는 작동하지만, 배경 이미지 대신 그라데이션 배경이 표시됩니다.
